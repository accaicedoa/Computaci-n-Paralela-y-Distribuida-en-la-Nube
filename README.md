## Descripción del proyecto

Este proyecto corresponde al Laboratorio III sobre computación en la nube, despliegue automático, plataformas PaaS y funciones Serverless.

El objetivo principal fue desplegar una página web sencilla en Vercel utilizando GitHub como repositorio de código fuente. Además, se implementó una función Serverless mediante una API en JavaScript, la cual responde en formato JSON y permite recibir parámetros por medio de la URL.
